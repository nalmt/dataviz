Credits: a brief overview

Datasets:

- NFA 2019: 

Is the National Footprint and Biocapacity Accounts data set (1961-2016) Prodvided by  "footprintnetwork"(https://www.footprintnetwork.org/resources/data/)

 We performed some data processing operations on the dataset  using csvkit and knime :  

- NFA 2019 have been merged with a [GFN Country Code Concordance Table](https://github.com/nalmt/dataviz/blob/main/datasets/GFN%20Country%20Code%20Concordance%20Table.csv). 

 - The continent of each country have been added.

 - Countries that no longer exist have been deleted.  

- The following columns have been deleted : country_code, ISO Alpha-2 Code, ISO Alpha-3 Code, QScore. 

- Some of the countries do not have crop_land, grazing_land, forest_land, fishing_ground, built_up_land or carbone hence these fields are set to 0 when the values are missing.  



Libraries and codes :

- We used theJavaScript library D3.js   

- The code for the treetop view is based on this zoomable treemap tutorial: https://observablehq.com/@d3/zoomable-treemap

Limits and issues :

- Limit: the dataset doesn't have all the values for every country and some values are not very accurate (having a bad data quality score)

- Issue: The issue with the treemap is when the difference between two variables values is big, the rectangles representing the smaller value is almost not visible.

- The website needs more tweaking to be more presentable.
